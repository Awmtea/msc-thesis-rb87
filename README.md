# msc-thesis-rb87
Python simulations accompanying my MSc thesis on a single ⁸⁷Rb hyperfine qubit in an optical tweezer. Includes Breit–Rabi clock-transition analysis, QuTiP Lindblad master-equation simulations of T₁/T₂/T₂* dynamics and Hahn echo, and a single-qubit gate error budget benchmarked against recent tweezer-array experiments.
